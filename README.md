# ระบบบันทึกการเติมน้ำมัน (Fuel Tracking System)

แอปพลิเคชันเว็บแบบง่ายสำหรับบันทึกและติดตามการเติมน้ำมันของรถในองค์กร รองรับการใช้งานทั้งบนคอมพิวเตอร์และมือถือ

## คุณสมบัติเด่น
- 🕒 **ลงเวลาอัตโนมัติ**: บันทึกวันและเวลาที่ทำรายการให้อัตโนมัติ (แก้ไขไม่ได้เพื่อความถูกต้อง)
- 🚗 **รองรับหลายทะเบียน**: ระบุทะเบียนรถและชื่อพนักงานผู้ใช้รถได้
- 📸 **แนบรูปสลิป**: สามารถถ่ายรูปหรือเลือกรูปสลิปน้ำมันเพื่อเก็บเป็นหลักฐานได้
- 📊 **สรุปผล**: แสดงค่าใช้จ่ายรวม จำนวนลิตร และราคาเฉลี่ยต่อลิตร
- 🛡️ **ระบบผู้ดูแล (Admin)**: 
    - ดูสถิติแยกตามรายทะเบียนรถและรายพนักงาน
    - ค้นหาและกรองข้อมูล
    - สิทธิ์ในการลบรายการที่ผิดพลาด
    - รหัสผ่าน Admin เริ่มต้น: `admin1234`
- 💾 **การจัดการข้อมูล**: 
    - ส่งออกข้อมูลเป็นไฟล์ CSV (สำหรับเปิดใน Excel)
    - ส่งออกข้อมูลเป็นไฟล์ JSON (สำหรับสำรองข้อมูล)
    - ข้อมูลทั้งหมดจัดเก็บอยู่ในเครื่องของผู้ใช้ (Local Storage) ไม่ต้องมี Server

## วิธีการใช้งาน
1. เปิดไฟล์ `index.html` ด้วยเว็บเบราว์เซอร์
2. กรอกข้อมูล ชื่อพนักงาน, ทะเบียนรถ, เลขไมล์, จำนวนลิตร และราคา
3. แนบรูปภาพสลิป (ถ้ามี)
4. กดปุ่ม "บันทึกข้อมูล"

## การเชื่อมต่อกับ AppSheet
หากต้องการใช้ร่วมกับ AppSheet เพื่อการทำงานที่ซับซ้อนขึ้น:
1. ใช้หัวข้อคอลัมน์จากไฟล์ `fuel_tracking_template.csv`
2. สร้าง Google Sheets และนำหัวข้อไปวางในแถวแรก
3. เปิด AppSheet และเลือก "Create app from data"
4. ตั้งค่าคอลัมน์ `Timestamp` เป็นประเภท `DateTime` และ `Initial Value` เป็น `NOW()`
5. ตั้งค่าคอลัมน์ `Slip Image` เป็นประเภท `Image`
